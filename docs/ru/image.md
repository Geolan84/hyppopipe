## Описание

`Image` используется для представления изображения внутри фреймворка.

При помощи метода `from_path` в память может быть загружено изображение в форматах: .jpg (.jpeg), .png, .tif (.tiff) и .dcm. По умолчанию загружаемые файлы проверяются на соответствие допустимым MIME-типам - невалидные файлы игнорируются.

Чтобы просмотреть изображение, достаточно использовать метод show().
